JellyTrack

Sends playback events and media metadata to JellyTrack for analysis.

Target URL. Examples: http://192.168.1.100:3000/api/plugin/events
The API key generated in JellyTrack settings (format: jt_xxxxxxxxxxxx)
Interval between each heartbeat sent to JellyTrack (default: 600 seconds)
Interval while playback is active (default: 5 seconds)
Interval while playback is paused (default: 30 seconds)
Minimum position jump reported as a manual seek (default: 20 seconds)
Maximum events kept locally while JellyTrack is unavailable (default: 500)
Maximum events resent per flush when JellyTrack responds again (default: 50)
Select the UI language used by JellyTrack. Leave on 'Auto' to use Jellyfin's server language.
