JellyTrack

Configure la connexion vers votre serveur JellyTrack.

URL cible exacte. Exemples : http://192.168.1.100:3000/api/plugin/events ou http://192.168.1.100:3000/api/webhook/jellyfin. Si vous mettez seulement l'hôte, /api/plugin/events est utilisé par défaut. Si Jellyfin tourne en Docker, n'utilisez pas localhost: utilisez l'IP hôte, host.docker.internal ou le nom de service Docker.
La clé API générée dans les paramètres de JellyTrack (format: jt_xxxxxxxxxxxx)
Intervalle entre chaque heartbeat envoyé à JellyTrack (défaut: 600 secondes)
Intervalle entre chaque mise à jour de progression de lecture (défaut: 15 secondes)
Sélectionnez la langue de l'interface utilisée par JellyTrack. Laisser sur «Auto» pour appliquer automatiquement la langue du serveur Jellyfin.
